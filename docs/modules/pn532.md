# PN532 — NFC / RFID 13.56 MHz

## Role

A 13.56 MHz NFC/RFID front-end that **reads and writes** contactless cards and tags. In NexusPi it handles dumping, analysing, and (for cards you own) cloning to rewritable "magic" cards.

- **Frequency:** 13.56 MHz
- **Supports:** MIFARE Classic 1K/4K, MIFARE Ultralight, NFC Type 2/4, ISO14443-A/B
- **Interface:** I²C (also SPI / UART via DIP switches)

## Wiring

I²C bus — see [`../wiring.md`](../wiring.md). Summary: VCC→3.3 V (Pin 1), GND→Pin 6, SDA→GPIO2 (Pin 3), SCL→GPIO3 (Pin 5). Set the board's DIP switches to **I²C**.

## Kali setup

Enable I²C and install libnfc:

```bash
sudo raspi-config        # Interface Options → I2C → Enable
sudo apt install libnfc-bin libnfc-dev i2c-tools
```

Tell libnfc how the PN532 is connected. Create `/etc/nfc/libnfc.conf`:

```
device.name = "PN532 over I2C"
device.connstring = "pn532_i2c:/dev/i2c-1"
```

Confirm the chip answers on the bus:

```bash
i2cdetect -y 1          # PN532 usually shows at address 0x24
nfc-list                # should report the PN532 and any card present
```

## Core commands

| Goal | Command |
|---|---|
| List reader + detect card | `nfc-list` |
| Read a MIFARE Classic dump | `nfc-mfclassic r a dump.mfd` |
| Write a dump to a card | `nfc-mfclassic w a dump.mfd` |
| Poll / identify a tag | `nfc-poll` |
| Read NDEF (NFC tags) | `nfc-mfultralight r tag.dump` |

For scripting, the Python library `nfcpy` is convenient and is what `backend/modules/nfc.py` wraps.

## Example — back up and clone your own card

```bash
# 1. Read your own card to a file
nfc-mfclassic r a my_card.mfd

# 2. Write it onto a blank magic card (UID-writable)
nfc-mfclassic w a my_card.mfd
```

> Standard MIFARE Classic cards have a locked UID. To clone the UID as well, you need a **"magic" / UID-rewritable** card.

## Limits & legal

- Only read/write/clone **cards you own** (your office badge, your hotel key, blank tags).
- Cloning bank cards, transit passes, or third-party access badges is illegal and explicitly out of scope — see [`threat-model.md`](threat-model.md).
- MIFARE Classic uses the broken Crypto-1 cipher; treat any "security" demonstration as educational, on your own credentials.
